# `update_scaffold`

Update


## Description

Update the scaffolded code to a new version of
 leprechaun.


## Usage

```r
update_scaffold(force = FALSE)
```


## Arguments

Argument      |Description
------------- |----------------
`force`     |     Force update, ignore safety checks.


## Details

This reads the content of the `.leprechaun` lock file
 and updates it too.
